# Media files

Add your media here. Filenames must match the paths in `lib/config.ts`.

## Required

| File            | Used for                                   |
| --------------- | ------------------------------------------ |
| `professor.jpg` | The professor photo at the top of the page |
| `videos/*.mp4`  | One video per language button (see below)  |

## Professor photo

Save the portrait here as `professor.jpg` (a portrait image, e.g. 600 x 800,
looks best). To use a different name, update `professorPhoto` in `lib/config.ts`.

## Videos (one per button)

Put the four button videos in the `videos/` subfolder — see `videos/README.md`
for the exact filenames.
